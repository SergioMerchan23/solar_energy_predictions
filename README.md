Descripcion:
Desarrolle un modelo para clasificar ciudades, paises o regiones segun su nivel de adopcion a sistemas de energia solar. Seran clasificados en "Alto", "Bajo", "Medio-Alto" y "Medio-Bajo", segun datos Geograficos, cantidad de horas anuales de luz solar, picos diarios de luz, entre otros datos. 

Tecnoligia y librerias utilizadas:
Utilice Python y las librerias pandas, sklearn, numpy, matplotlib y seaborn. Ademas utilice el modelo de RandomForest para la clasificacion

Responsabilidad clave:
Ademas de la limpieza de datos, se realizo una EDA (analisis exploratorio de datos), se creo el modelo RandomForest para clasificar la adopcion de sistemas de energia solar.

Impacto logrado:
Se obtuvo un modelo clasificatorio con una precision promedio de 90%. 
Para clasificacion de regiones con "Alta" adopcion a sistemas energeticos fue - Precision: 0.75 - Recall: 1.00 - F1-score: 0.86
Para clasificacion de regiones con "Baja" adopcion a sistemas energeticos fue - Precision/Recall/F1: 1.00
Para clasificacion de regiones con "Media - Alto" adopcion a sistemas energeticos fue - Precision/Recall/F1: 1.00
Para clasificacion de regiones con "Media - Bajo" adopcion a sistemas energeticos fue - Precision: 1.00 - Recall: 0.67 (2 de 3 correctos)
